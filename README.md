#  Métodos Numéricos — Implementações em Octave

Este repositório contém implementações desenvolvidas durante a disciplina de **Métodos Numéricos**, com foco em **Álgebra Linear Numérica**.

O objetivo é aplicar conceitos teóricos por meio de algoritmos implementados no **Octave**, facilitando o entendimento prático de operações com matrizes e vetores.

---

## 📌 Conteúdos abordados

* ✔ Operações com matrizes
* ✔ Eliminação de Gauss
* ✔ Método de Gauss-Jordan
* ✔ Decomposição LU
* ✔ Decomposição de Cholesky
* ✔ Projeção escalar e vetorial
* ✔ Complemento ortogonal
* ✔ Ortogonalização de Gram-Schmidt
* ✔ Decomposição GR (matriz ortogonal + triangular)
* ✔ Decomposição QR (matriz ortonormal + triangular)

---

##  Objetivo

Este repositório foi criado com o intuito de:

* Consolidar o aprendizado dos conceitos de Álgebra Linear
* Implementar algoritmos fundamentais manualmente
* Entender o funcionamento interno de métodos numéricos
* Servir como material de estudo para provas e revisões

---

## ⚙️ Tecnologias utilizadas

* **GNU Octave**
